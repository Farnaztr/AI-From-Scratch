# AI From Scratch

A collection of Machine Learning and Deep Learning algorithms implemented **from scratch** using only NumPy and Python.

The goal of this repository is to deeply understand how AI algorithms work internally instead of relying on high-level libraries.

## Current Implementations

- ✅ Linear Regression (Gradient Descent)

## Planned Implementations

- Logistic Regression
- K-Nearest Neighbors (KNN)
- Decision Tree
- Naive Bayes
- Perceptron
- Neural Network
- Convolutional Neural Network (CNN)
- Recurrent Neural Network (RNN)
- Transformer
- Attention Mechanism

## Technologies

- Python
- NumPy
- Matplotlib


## Purpose

This repository is created for learning and educational purposes.
Every algorithm is implemented from scratch to understand the underlying mathematics and optimization process.
